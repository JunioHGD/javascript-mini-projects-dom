# Product Catalog DOM

Mini-projeto desenvolvido com HTML, CSS e JavaScript para praticar manipulação do DOM, renderização dinâmica de produtos e filtro por categoria.

O projeto simula um catálogo simples de produtos de tecnologia, onde os itens são exibidos na tela a partir de um array de objetos.

## Conceitos praticados

- HTML semântico
- CSS responsivo
- JavaScript DOM
- Arrays de objetos
- `.filter()`
- Eventos com `addEventListener`
- Renderização dinâmica
- Criação de elementos com `createElement`
- Limpeza de container com `replaceChildren()`

## Funcionalidades

- Exibição dinâmica dos produtos
- Filtro por categoria
- Atualização da lista ao selecionar uma categoria
- Cards com imagem, nome, categoria, preço e status
- Layout responsivo
- Separação entre HTML, CSS e JavaScript

## Como funciona

1. Os produtos são armazenados em um array de objetos.
2. O JavaScript cria os cards dinamicamente.
3. O usuário seleciona uma categoria no filtro.
4. A lista de produtos é filtrada com base na categoria escolhida.
5. O conteúdo exibido na tela é atualizado sem recarregar a página.

## Aprendizado

Este projeto marcou minha transição dos exercícios no console para uma interface real com HTML, CSS e JavaScript.

Durante o desenvolvimento, pratiquei a conexão entre lógica de programação e interface, usando dados em arrays, filtros, eventos e manipulação direta do DOM.
