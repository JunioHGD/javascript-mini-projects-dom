# JavaScript Mini Projects DOM

Este repositório reúne mini-projetos desenvolvidos com HTML, CSS e JavaScript, com foco em manipulação do DOM e construção de interfaces simples.

A proposta é praticar a conexão entre a lógica em JavaScript e a interface da página, evoluindo aos poucos de exercícios no console para projetos visuais e interativos.

## Objetivo

O objetivo deste repositório é registrar minha evolução prática no desenvolvimento front-end, aplicando conceitos como:

- estruturação com HTML
- estilização com CSS
- manipulação do DOM
- eventos de usuário
- renderização dinâmica de dados
- filtros e interações simples
- organização de arquivos
- responsividade básica

## Estrutura dos projetos

Cada mini-projeto fica em sua própria pasta, contendo normalmente:

nome-do-projeto/
├── index.html
├── style.css
└── script.js

Essa organização ajuda a manter cada exercício isolado, facilitando testes, manutenção e futuras melhorias.

## Projetos

Exemplo de projeto presente neste repositório:

product-catalog-dom/

Um catálogo de produtos com filtro por categoria, renderização dinâmica de cards e uso de JavaScript para manipular os dados exibidos na tela.

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## Aprendizado

Este repositório faz parte da minha prática contínua em desenvolvimento web.

A ideia não é criar projetos grandes de início, mas sim consolidar fundamentos importantes para depois evoluir para aplicações mais completas, com código mais organizado, interfaces melhores e regras de negócio mais próximas de situações reais.
